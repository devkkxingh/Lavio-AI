Translation Dropdown Test Page

Instructions:

  1. Load this extension in Chrome (chrome://extensions/)
  2. Enable Developer mode and click "Load unpacked"
  3. Select the hackathon folder
  4. Refresh this page
  5. Look for the floating Lavio button
  6. Select any text below and click the translate button to test the dropdown
